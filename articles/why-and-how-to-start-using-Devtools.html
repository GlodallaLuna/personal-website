Why and how to start using Devtools if you are learning CSS

Sep 22, 2022

If I could give a piece of advice to my former self back when I started learning web development, it would be to start using Devtools from day one. In my personal experience, while the first CSS courses I followed online taught me a lot about CSS theory and encouraged me to learn through practice, they didn’t put enough emphasis on the importance of developer tools. In JavaScript, on the other hand, every lesson starts by showing you the console, as it’s crucial to understanding how the language works. I believe that a good understanding of what happens behind the scenes on a webpage is not only useful for debugging, but is also a fundamental step in your journey to really understand CSS.
